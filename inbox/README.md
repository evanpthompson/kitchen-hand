# inbox/

Raw, unmodified captures — one subfolder per candidate recipe, named with the
same slug the recipe will eventually use. See `docs/ingestion.md` for what to
put here per source type (URL, document, Instagram, TikTok, pasted text).

Nothing in here is a finished recipe. Don't hand-edit these into recipe
format in place — normalize into `recipes/_drafts/<slug>.yaml` instead, and
keep the original capture here as the audit trail.
